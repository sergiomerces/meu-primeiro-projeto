Tag html

A tag html é usada para marcar o início e o fim de um documento.

[Voltar]

Tag head

A tag head funciona como um cabeçalho do documento html, ela abriga o título da página que vai aparecer na aba do navegador e outras informações de metadados invisíveis aos usuários.

[Voltar]

Tag title

A tag title define o título do documento que irá aparecer na aba do navegador.

[Voltar]

Tag body

A tag body delimita o corpo do documento html que será visível ao usuário no navegador.

[Voltar]

Tag heading (h1 ... h6)

As tags h1 .... h6 são tags próprias para marcação de títulos e o número que as acompanha escala a sua importância dentro do fluxo do texto.

Exemplo:

Título h1

Título h2

Título h3

Título h4

Título h5
Título h6

[Voltar]

Tag hr

A tag hr desenha uma linha horizontal na tela, pode ser usada para separar seções.

Tag strong

A tag strong aplica o estilo negrito ao texto, usado para dar ênfase ou destaque.

texto em negrito usando strong

[Voltar]

Tag ins

A tag ins aplica o estilo sublinhado ao texto, também usado para destacar palavras-chaves.

texto sublinhado usando ins

[Voltar]

Tag em

A tag em aplica o estilo itálico ao texto, usado para se referir a termos estrangeiros, citações e fontes.

texto em itálico usando em

[Voltar]

Tag del

A tag del aplica o estilo tachado, marcando um trecho de texto suprimido de sua publicação original

texto a ser mostrado como tachado usando del

[Voltar]

Tag sup

A tag sup aplica o estilo superescrito ao texto, usado para grafia de números exponenciais em fórmulas matemáticas.

texto em superescrito usando sup a2 = b2 + c2

[Voltar]

Tag sub

A tag sub aplica o estilo subscrito ao texto, usado para grafia de números de índice em equações matemáticas e fórmulas químicas.

texto em subscrito usando sub X1 = 0 e X2 = 3

[Voltar]

Tag mark

A tag mark aplica o estilo marcação ao texto, usado para destacar trecho no texto para fácil leitura.

texto em destacado usando mark

[Voltar]

Tag a

A tag a cria uma ligação (link) entre um ponto de ancoragem dentro do texto e outra seção ou entre um documento e outro do mesmo website ou outro.

website: sergiomerces.github.io

[Voltar]

Tag blockquote

A tag blockquote aplica um conjunto recuos maiores ao texto, usado para marcar um pensamento ou citação.

Todo grande problema que você acredita não poder resolver, tem problemas menores que você pode separálos e resolvê-los.

[Voltar]

Tag ol

A tag ol marca o início e o fim de uma lista ordenada.

  1. HTML
  2. CSS
  3. Javascript

[Voltar]

Tag ul

A tag ul marca o início e o fim de uma lista não ordenada.

  • Git
  • Bootstrap
  • Angular

[Voltar]

Tag li

A tag li cria um item dentro de uma lista ordenada ou não ordenado, como os demonstrados acima.

[Voltar]

Tag font

A tag font define o tipo, tamanho e cor, embora ainda seja entendido por grande maioria dos navegadores, caiu em desuso com o avanço do CSS.

fonte definida com tag font

[Voltar]

Tag abbr

A tag abbr define termos e siglas usados no texto com uma etiqueta pop over.

marcação HTML para definição de termos

[Voltar]

Tag p

A tag paragrafo faz a marcação de início e fim de um parágrafo.
